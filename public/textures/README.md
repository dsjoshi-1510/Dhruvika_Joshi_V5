# Textures Folder

Place your custom texture images here!

## How to Add Textures

1. **Download texture images** from:
   - Unsplash: https://unsplash.com/s/photos/texture
   - Pexels: https://www.pexels.com/search/texture/
   - Pixabay: https://pixabay.com/images/search/texture/
   - Texture.com: https://www.textures.com/

2. **Save images here** with clear names:
   - `vintage-paper.jpg`
   - `linen-fabric.png`
   - `grain-texture.jpg`

3. **Add CSS** in `src/index.css` (see CUSTOM_TEXTURE_GUIDE.md)

4. **Use in components**:
   ```tsx
   <div className="grainy-texture texture-your-name">
   ```

## Image Requirements

- **Format**: JPG or PNG
- **Size**: 500x500px to 2000x2000px (square works best)
- **File size**: Under 500KB (optimize if needed)
- **Style**: Seamless/tileable textures work best

See `CUSTOM_TEXTURE_GUIDE.md` for detailed instructions!
